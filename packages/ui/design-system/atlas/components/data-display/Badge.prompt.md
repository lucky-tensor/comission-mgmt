Compact label for record status, category, or counts. Use `dot` for live state (Active / Paused / Failed).

```jsx
<Badge tone="success" dot>Active</Badge>
<Badge tone="warning" dot>Pending</Badge>
<Badge tone="danger">Suspended</Badge>
<Badge>v2.4.1</Badge>
```

Tones: `neutral` · `info` · `success` · `warning` · `danger`. Keep labels to one or two words.
